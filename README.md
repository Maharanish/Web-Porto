# Maha's Portfolio

This is a personal portfolio website showcasing my projects, skills, and experiences in technology development such as web development, software development, and data analysis. 

## How to Run

Follow these steps to run the portfolio website on your local machine:

1. Clone this repository:

    ```bash
    git clone (https://github.com/Maharanish/Web-Porto)
    ```

2. Install Live Server globally:

    ```bash
    npm install -g live-server
    ```

3. Run Live Server in the project directory:

    ```bash
    live-server FE
    ```

This will start a local server and open the portfolio website in your default web browser.

## Technologies Used

- HTML
- CSS (Tailwind CSS)
- JavaScript
- Live Server
