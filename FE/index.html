NomNom : Meal Scheduler

NomNom is a user-friendly meal planning app that allows efficient scheduling, detailed recipe viewing, and seamless shopping list management. It offers offline access and requires no authentication, providing convenient flexibility for diverse meal planning needs. I contributed to designing the UI and integrating the backend with the database. If you want to explore the project further, you can click directly on the project title.

Farm House Game

This Java-based farm management application uses JavaFX for its graphical user interface (GUI), supporting gameplay for two players to plant crops, raise animals, sell products, handle item cards, defend against bear attacks, and aim to earn the most money in 20 turns. It includes save & load functionality in txt format and supports additional file formats via a plugin. I contributed to integrating UI elements with the backend and provided backend support during development. If you want to explore the project further, you can click directly on the project title.
